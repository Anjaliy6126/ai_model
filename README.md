# 🏠 House Price Prediction

## About the Project

This is a simple Machine Learning project that predicts the price of a house based on its area (in square feet). I built this project using Python, Scikit-learn, and Streamlit to understand how Linear Regression works and how to deploy a machine learning model as a web application.

## Features

- Predicts house prices based on house area
- Simple and user-friendly interface
- Uses Linear Regression for prediction
- Displays model coefficient and intercept

## Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn

## Project Structure

```
ai_model3/
│── streamlit_app.py
│── houseprice.csv
│── requirements.txt
│── README.md
```

## Dataset

The dataset contains house prices and their corresponding area in square feet. It is used to train the Linear Regression model for prediction.

- How to build a Linear Regression model using Scikit-learn
- How to create a web application using Streamlit
- How to deploy a Streamlit app using Streamlit Community Cloud
- How to upload and manage projects using GitHub
